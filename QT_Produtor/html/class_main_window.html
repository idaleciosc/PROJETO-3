Descrição Detalhada

The MainWindow class permite criar uma janela e mostrar componentes na tela.

A partir dela, permite realizar as funcionalidades do programa como: conectar e desconectar do servidor, e iniciar e parar a produção de dados num determinado intervalo de tempo estipulado pelo usuário.
